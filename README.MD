# Shelfie app

This app was created with
```sh
npx create-expo-app@latest --template blank-typescript
```

## Setup

- Use node >= 22.9
- Use JDK >= 17

## Build

### Locally for android
```sh
npm android 
```

**Run:**
```sh
npm start
```
